# EEG Analysis

## Get Started

### using Linux or MacOS
install the environment:
```
git clone --depth 1 https://github.com/ZhangJiayiLab/EEGAnalysis.git
conda env create -f environment.yml -n eeg
source activate eeg
python setup.py isntall
```

### using Windows
install the environment:
```
git clone  --depth 1 https://github.com/ZhangJiayiLab/EEGAnalysis.git
conda env create -f environment.yml -n eeg
activate eeg
python setup.py install
```
